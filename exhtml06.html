구글 검색 :


위 input에 검색어를 입력하고 submit 버튼을 클릭하면 구글 검색 결과 출력
1. submit 버튼에 표시되는 글자를 "검색" 으로 변경
2. 검색 버튼을 클릭하면, https://www.google.com/search?q="검색어"
와 같은 주소로 이동을 하여, 구글 검색 결과 페이지가 출력되도록
3. 검색 결과과 새 창으로 출력되도록 변경
4. lr=lang_ko 파라미터를 추가하면 한국어 검색 결과만 나온다.
영어, 한국어, 일본어 중에 하나를 선택하여 검색 할 수 있도록 셀렉트
5. 다음 검색으로도 한 세트 더 추가
6. 구글 검색 결과가 iframe에 나오도록 변경
